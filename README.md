📂 Programs Included
1️⃣ Armstrong Number Program

File: Armstrong.java

This program checks whether a number is an Armstrong number.

Example:

Input: 153
Output: Armstrong Number

Explanation:
153 = 1³ + 5³ + 3³ = 153

Concepts used:

Loops

Mathematical operations

Number logic

2️⃣ Basic Java Arrays Program

This program demonstrates basic array operations in Java.

Concepts covered:

Declaring arrays

Initializing arrays

Accessing array elements

Iterating arrays using loops

Example:

int arr[] = {10,20,30,40};

3️⃣ Factorial Program

File: Factorialprogram.java

This program calculates the factorial of a given number.

Example:

Input: 5
Output: 120

Explanation:

5! = 5 × 4 × 3 × 2 × 1 = 120

Concepts used:

Loops

Multiplication logic

Mathematical computation

4️⃣ Fibonacci Series

File: Fibonacci.java

This program prints the Fibonacci sequence.

Example:

Input: 5
Output:
0 1 1 2 3

Explanation:
Each number is the sum of the previous two numbers.

Concepts used:

Looping

Variable swapping

Logical sequence generation

5️⃣ Palindrome Number

File: PalindromeNumber.java

This program checks whether a number is a Palindrome.

Example:

Input: 121
Output: Palindrome

Explanation:
A palindrome number reads the same forward and backward.

Examples:

121
1331
4554

Concepts used:

Reverse number logic

Loops

Conditional statements
